# my-gpt

The purpose of this app is to build something similar to chatGPT.

This App uses the OpenAi api. To make this application work perfectly on your system, you will have to get an OpenAi api key. The steps to get your own api key are given below:

1. You go to the OpenAi home page "https://openai.com/" and then "sign up".
2. Click on API.
3. Almost at the top right corner of your browser,click on "Personal".
4. Click on "View Api Keys".
5. Click on "create new secret key".

When you have gotten your api key, go to the ".envSample" file of this project and set the Open api key as the value for VUE_APP_API_KEY.
